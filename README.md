# SpaceLoader

### 介绍
我的世界将模组转换为DLL的技术, 它还有个名字叫 "Loader"<br>
注意 ! 它只能加载 ".class" , 并不能不能将 "resources" 中的文件加载，因此无法使用"mixin"<br>
A minecraft forge mod injector<br>

### 源码介绍
Loader [核心]<br>
基于火山软件开发平台(64位)开发。

JavaDllPacker<br>
基于Idea2024 JDK22开发。
